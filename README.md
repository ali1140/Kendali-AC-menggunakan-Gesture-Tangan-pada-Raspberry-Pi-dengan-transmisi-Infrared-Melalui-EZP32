## 📖 Deskripsi Singkat
Sistem kendali pintar menggunakan pengenalan gestur tangan (Gesture Recognition) yang dijalankan pada Raspberry Pi.

Project ini merupakan bagian dari portofolio dan eksperimen pengembangan perangkat lunak/keras. Repository ini berisi *source code* dan dokumentasi dasar untuk memahami alur kerja dan implementasi sistem.

## 🛠️ Teknologi yang Digunakan
- **Utama:** Python, Raspberry Pi, Machine Learning, Infrared Transmission
